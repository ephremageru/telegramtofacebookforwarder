# Telegram to Facebook Forwarder

This project provides a seamless way to forward messages from Telegram to Facebook.

## Installation

To install the project, follow these steps:

1. Clone the repository:
   
   ```bash
   git clone https://github.com/ephremageru/telegramtofacebookforwarder.git
   ```

2. Install the required packages:
   
   ```bash
   npm install
   ```

## Usage

To use this utility, run the following command:
   
   ```bash
   node index.js
   ```

## Features
- Forward messages from Telegram to Facebook.
- Supports multiple message types and media.

## Configuration

Before running the utility, configure the `config.json` file with the necessary API keys and settings.
